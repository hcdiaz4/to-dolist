
# To Do List

"To Do List" es una aplicación simple y eficaz para gestionar tus tareas diarias. Crea listas, agrega tareas, establece prioridades y marca como completadas. Mantén tu vida organizada y aumenta tu productividad.




## Installation

- Clona el repositorio

```bash
  git clone git@github.com:hcdiaz4/to-dolist.git
  cd nequi-test
```

- Instala las dependencias:
```bash
  npm install
```
Se recomienda usar versión LTS de Node JS.



## Uso
Una vez instalado hayas clonado e instalado las dependencias del proyecto podras ejecutar los siguientes comandos sobre el directorio raíz (/nequi-test):

1. Para ejecutar la app en tu local ejecuta el comando **_ionic serve_**
2. **_npm run start:build:[platform]_** para contruir tu proyecto en la plataforma que selecciones (android, ios).
```
  npm run start:build:android
  npm run start:build:ios
```
3. **_npm run open:[platform]_** para abrir el ID de desarrollo de cada plataforma (Xcode, Android Studio).
```
  npm run open:android
  npm run open:ios
```


## Tecnologías usadas

- Ionic v7.2
- Angular v18.2.7
- Capacitor v6.1.2
- Angular Fire v18.0.1 (Realtime Database, Remote Config)
